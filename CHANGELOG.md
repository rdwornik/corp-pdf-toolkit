# Changelog — corp-pdf-toolkit

## [0.4.0] — 2026-03-15

- PDF-to-Markdown extraction with TOC detection (bookmarks + dot-leader parsing)
- Chapter and page range selection
- PDF anonymization via local Ollama LLM (async, parallel chunk processing)
- Shared utilities: TOC parsing, page selection, text extraction, file resolution

## Status

Standalone utility with minimal use. Candidate for archival or merge into
corp-knowledge-extractor if PDF extraction needs grow.
